# keykeyv2 제작 가이드

---

## 키보드 특징

- **스플릿 키보드** (좌/우 분리형)
- **각 사이드 5행 6열** = 총 60키 (매트릭스)
- **왼쪽 사이드가 마스터**
- **방향키 4개**(다이렉트 핀, 5방향 네비게이션 스위치)
- **캡스락 LED** 1개
- **Vial-QMK** 사용 (GUI로 키맵 변경 가능)

## 납땜

### 1. 납땜 위치
- 다이오드, 핫스왑 소켓 후면에 장착
- RP2024-zero, PJ-320A TRRS 잭, LED 앞면 장착

### 2. 다이오드 방향
- (|) 표시가 있는 방향이 양 사이드의 가운데를 향해야 함 (왼쪽 사이드에서는 오른쪽, 오른쪽 사이드에서는 왼쪽)


---

## 필요 프로그램 설치

### 1. 파일 다운로드
/last/keykeyv2_vial.uf2 다운로드

### 2. 각 사이드 마다 RP2024-zero 펌웨어 업로드
- 1. 모든 선 분리 후 USB만 단독으로 연결 (꼭 통신 케이블용 사용)
- 1. BOOT 버튼, RESET 버튼 동시에 누름
- 2. 1초 후 RESET 버튼에서 손때고 그 다음 1초 후 BOOT 버튼에서 손 땜
- 3. 외장 드라이버로 인식되면 keykeyv2_vial.uf2 파일 복사해서 저장
- 4. 자동으로 RP2024-zero 재부팅

---

## 작동 테스트

1. USB가 연결되지 않은 상태에서 TRRS선 연결
> ⚠️ USB가 연결된 상태에서 TRRS선 분리하면 고장남
2. USB 왼쪽 사이드에 연결
> 항상 왼쪽이 마스터
